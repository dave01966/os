1. Click the download link above

## ✨ Features

- **Desktop** with clickable icons and right-click context menu
- **Taskbar** with Start menu, open app buttons, and a live clock
- **Window management** — drag, resize, minimize, maximize, and close windows
- **6 built-in apps**

---

## 📦 Apps

| App | Description |
|-----|-------------|
| 📝 Notepad | Simple text editor |
| 🔢 Calculator | Full arithmetic calculator |
| 📁 File Manager | Browse a simulated file system |
| 🖥️ Terminal | Command-line shell with real commands |
| 🌐 Browser | Embedded iframe browser |
| ⚙️ Settings | Change wallpaper and toggle system options |

---

## 🖥️ Terminal Commands

| Command | Description |
|---------|-------------|
| `help` | List all available commands |
| `echo [text]` | Print text to the terminal |
| `date` | Show current date and time |
| `clear` | Clear the terminal screen |
| `calc [expr]` | Evaluate a math expression |
| `ls` | List files |
| `whoami` | Show current user |
| `uptime` | Show system uptime |
| `neofetch` | Display system info |
| `color [0-5]` | Change the desktop wallpaper |

No build tools. No npm. No frameworks.

## 🎨 Customization

- **Wallpapers** — open Settings and pick from 6 gradient themes, or edit the `WALLS` array in the JS to add your own
- **Apps** — add new entries to the `APPS` array and implement `buildContent()` for them
- **Colors** — all UI colors are defined in the `<style>` block at the top of the file
